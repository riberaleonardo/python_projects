Project 3: Full-Stack Data Web Application

Project Overview

This web application combines a structured train transportation dataset with live weather data from an external API into one unified Django system.

The core dataset contains train trip records across Spain, including origin and destination cities, departure and arrival times, train type, pricing, and fare categories.

The API integration connects directly to this dataset through the same cities used in the train records. Current weather is fetched for those cities and stored in the database, allowing the application to display live city conditions alongside train trip information.

Navigation

  • View All Records
  • Create New Record

To edit or delete records, navigate to the records list and select a specific entry.
