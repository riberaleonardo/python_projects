Project 3: Full-Stack Data Web Application

Project Overview

This web application integrates two prior projects into a single full-stack Django system. It combines a structured dataset from Project 1 with a live API data pipeline from Project 2.

Project 1: Train Dataset

The core dataset consists of train trip records across Spain, including origin and destination cities, departure and arrival times, train type, pricing, and fare categories.

This dataset is stored in a relational database using Django models and can be viewed, created, updated, and deleted through the application interface.

Project 2: API Integration

The application also includes a live data ingestion feature that fetches real-time data from an external API using a Django management command.

This demonstrates how external data sources can be integrated into an existing database using automated pipelines and ORM-based storage.

Navigation
